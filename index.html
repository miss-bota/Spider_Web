    Strategy Summary

    The system uses Kruskal's Algorithm. It first rejects anomalous/cursed edges, sorts valid edges by weight, then uses Union-Find to build a cycle-free MST.

    Cursed edges are detected using constraints: weight must be between 1 and 20, no self-loops, no duplicate connections, and no invalid node IDs.
